Vertical Mobile Navbar

1. Butona basınca ne oluyor? Navbar div’inin class’ına "responsive" ekleniyor veya çıkarılıyor. Bunu JavaScript toggleNavbar fonksiyonunda yapıyoruz.

2. Responsive şekilde nasıl yapılır? Media Query (@media) kullanıyoruz. Ekran 600px’den küçükse hamburger butonu gösteriliyor, linkler gizleniyor. Ekran büyükse (600px üstü) normal navbar gözüküyor, hamburger gizleniyor.

3. Style değiştirerek değil, class ekleyip çıkararak yapmak? Şu anda zaten class ekleyip çıkarıyoruz (responsive). Doğru yöntem bu. (Yani element.style.display = 'block' gibi doğrudan stil değiştirmek yanlış olurdu.)
